¿Que es ONLY ONE?

Only One es una pagina en la cual puedes buscar jugadores cuando a ti y a tus amigos les falte uno mas para esas partidas. La forma es simple, seleccionas el juego el cual quieres jugar y debes llenar con tus datos de jugador, id, nivel, etc. Ademas de seleccionar el modo de juego. Y listo! ya puedes encontrar a ese jugador faltante.

Y si tu eres el que quiere unirse a aquellos que les falte uno mas. Tambien puedes! Solo debes llenar con tus datos un mini cuestionario y tendras acceso a una lista de grupos a los cuales les faltas tu. Además, si gustas volver a buscar otro jugador, puedes registrarte y todos tus datos quedaran guardados.
